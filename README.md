# web-app

## Project setup

Copy/Paste `.env.example` file, rename the copy to `.env` and replace `VUE_APP_BACKEND_ENDPOINT` with your server adress

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```

### Run your unit tests

```
yarn run test:unit
```
